# Bill of Materials (BOM) – Swatchle

## Core Components
- **LilyGO T-Display S3**
  - ESP32-S3 microcontroller with integrated touchscreen and display  
  - https://lilygo.cc/products/t-display-s3?variant=42589373268149

- **DF Robot TCS34725 RGB Color Sensor (Gravity / Adafruit compatible)**
  - Used for real-world color sensing  
  - https://www.dfrobot.com/product-1546.html?srsltid=AfmBOookWUY95t35lH4IGewgszA3tHkXFTIchdyfdj55R_PjU7mW82LY

- **PKCell Li-Po Battery (3.7V, JST-PH 2.0)**
  - Power source for portable use  
  - Capacity: ~500 mAh
  - https://www.tinytronics.nl/index.php?route=product/product&product_id=5875

## Connectivity
- **WiFi & Bluetooth**
  - Integrated via ESP32-S3 module on T-Display S3

## Cables & Adapters
- **JST-PH 2.0 → JST-PH 1.0 adapter cable**
  - Required to connect battery to T-Display S3 (only supports JST-PH 1.0)

## Miscellaneous
- Hook-up wires (30 AWG)
- Heat shrink tubing (2–3 mm)
